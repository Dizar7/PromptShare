# PRD: PromptShare (글로벌 지식 자산화를 위한 AI 스킬 공유 플랫폼)

## 1. 프로젝트 개요 (Vision)
> **"국경 없는 노하우 공유, 모두를 위한 AI 기술 자산화"**

PromptShare는 개인의 로컬 환경이나 개별 저장소에 파편화되어 있는 AI 활용 노하우(Skill)를 전사적으로 통합하고 자산화하기 위한 글로벌 플랫폼입니다. 특히 Git이나 마크다운 작성이 익숙하지 않은 비개발 직군도 손쉽게 자신의 업무 노하우를 표준화된 'AI 스킬 파일'로 변환하고, 전 세계 동료들과 4개 국어로 공유할 수 있도록 돕습니다.

## 2. 배경 및 필요성 (Problem Statement)
- **지식의 파편화:** 개인이 가진 뛰어난 프롬프트 노하우가 서로의 로컬에만 머물러 있어 팀 간 지식 전달이 단절됨.
- **글로벌 협업의 한계:** 언어 장벽으로 인해 해외 지사의 우수한 프롬프트 사례를 국내 팀이 즉시 활용하기 어려움.
- **데이터 기반 의사결정 부재:** 어떤 부서에서 어떤 AI 스킬이 가장 많이 활용되는지 파악할 수 있는 통계 지표가 부족함.

## 3. 핵심 기능 (Key Features)

### 3.1 지능형 스킬 엔진 (AI-Powered Core)
- **AI 기반 스킬 정제:** Gemini 2.5 Flash API를 활용하여 거친 텍스트 초안을 정교한 시스템 프롬프트 형식으로 자동 최적화.
- **GitHub 자동 연동:** 생성된 스킬을 버튼 하나로 사내 공용 GitHub 저장소에 자동 커밋 및 푸시.

### 3.2 글로벌 확장성 (Global Scalability)
- **4개 국어 지원 (i18n):** 한국어, 영어, 일본어, 중국어(간체) 환경을 완벽하게 지원하여 글로벌 협업 환경 구축.
- **다국어 UI:** 실시간 언어 전환 시스템을 통해 모든 페이지의 정보를 사용자 언어에 맞게 제공.

### 3.3 데이터 인사이트 (Data & Insights)
- **실시간 통계 대시보드:** 부서별 스킬 등록 현황 및 누적 조회수를 시각화하여 사내 AI 활용 트렌드 분석.
- **지능형 추천 시스템:** 조회수 및 활용도 기반의 '인기 스킬(Recommended Skills)' 랭킹 제공.

### 3.4 보안 및 사용자 경험 (Security & UX)
- **JWT 기반 인증:** 안전한 회원가입 및 로그인 시스템을 통해 개인별 스킬 관리 및 프로필 최적화.
- **반응형 모던 UI:** Tailwind CSS를 활용한 프리미엄 다크 모드 및 사이드바 기반의 직관적인 대시보드 레이아웃.

## 4. 기술 스택 (Technical Stack)

| 구분 | 기술 스택 | 비고 |
| :--- | :--- | :--- |
| **Frontend** | React (Vite), Tailwind CSS | 모던 SaaS 스타일 UI / Responsive |
| **Backend** | FastAPI (Python) | 비동기 고성능 API 서버 |
| **Database** | PostgreSQL | 사용자 정보 및 스킬 메타데이터 관리 |
| **AI Engine** | Google Gemini 2.5 Flash | 프롬프트 정제 및 지침 최적화 |
| **VCS/Storage** | GitHub API | 스킬 파일 버전 관리 및 자동화 저장소 |
| **Auth** | JWT (JSON Web Token) | 보안 인증 및 상태 유지 |

## 5. 성과 및 가치 (Business Value)
- **생산성 향상:** 비개발 직군의 스킬 파일 작성 시간을 90% 이상 단축.
- **지식 자산화:** 개인의 노하우를 조직의 공용 자산으로 변환하여 리스크 관리 및 협업 효율 극대화.
- **데이터 대시보드:** 부서별 AI 도입 수준을 정량적으로 파악하여 교육 및 지원 전략 수립 가능.

## 6. 향후 로드맵 (Future Roadmap)
- **소셜 기능 강화:** 스킬별 댓글 및 좋아요 기능을 통한 커뮤니티 활성화.
- **프롬프트 템플릿 마켓:** 직무별 베스트 프롬프트 템플릿 세트 제공.
- **실시간 협업 편집:** 여러 사용자가 동시에 프롬프트를 다듬을 수 있는 협업 에디터 도입.
